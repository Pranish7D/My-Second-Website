Workshop 2: Ethical Hacking and Penetration Testing

About the Workshop:

This workshop focuses on ethical hacking and penetration testing techniques. Participants will learn how to assess and secure computer systems by identifying vulnerabilities and exploiting them within an ethical framework.

Requirements:

Basic knowledge of networking and computer systems. Familiarity with Linux command line is beneficial but not mandatory.

Certification Key:

Certified Ethical Hacker (CEH) - Level 1

Takeaways:

  • Understanding of ethical hacking principles
  • Proficiency in penetration testing methodologies
  • Hands-on experience with various hacking tools
  • Ability to identify and fix vulnerabilities

Target Audience:

  • IT professionals
  • Network administrators
  • Security enthusiasts
  • Aspiring penetration testers

Agenda:

  1. Introduction to Ethical Hacking
  2. Footprinting and Reconnaissance
  3. Scanning Networks
  4. Enumeration and System Hacking
  5. Malware Threats and Countermeasures
  6. Social Engineering Attacks
  7. Web Application Security
  8. Wireless Network Security
  9. Hands-on Lab Sessions
  10. Final Assessment and Q&A
