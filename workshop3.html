Workshop 3: Secure Coding Practices

About the Workshop:

This workshop emphasizes the importance of writing secure code to prevent vulnerabilities and data breaches. Participants will learn about common programming pitfalls, security libraries, and techniques to ensure software resilience.

Requirements:

Basic programming knowledge in a language such as Python, Java, or C/C++. Participants should bring their laptops for hands-on exercises.

Certification Key:

Certified Secure Code Developer

Takeaways:

  • Understanding of secure coding principles
  • Proficiency in identifying and mitigating common vulnerabilities
  • Hands-on experience with secure coding practices
  • Ability to write more secure software

Target Audience:

  • Software developers
  • Web developers
  • Application programmers
  • Tech leads

Agenda:

  1. Introduction to Secure Coding
  2. Common Vulnerabilities (SQL Injection, XSS, CSRF, etc.)
  3. Input Validation and Output Encoding
  4. Authentication and Authorization
  5. Secure Error Handling
  6. Secure Data Storage
  7. Security Libraries and Frameworks
  8. Secure Development Lifecycle
  9. Hands-on Coding Exercises
  10. Code Review Techniques
  11. Assessment and Wrap-up
